🌳 Curso de DOM para Juniors

Aprende cómo JavaScript "ve" y manipula una página web. Cada lección tiene teoría clara y demos interactivas con las que puedes jugar.

¿Qué vas a aprender?

  • Qué son el DOM y el BOM, y en qué se diferencian.
  • Cómo moverte entre elementos (padres, hijos y hermanos).
  • Cómo buscar elementos con getElementById, querySelector y más.
  • Qué son las propiedades de un nodo y la jerarquía Node → Element.
  • La diferencia entre atributos y propiedades, y cómo usar dataset y style.
💡 Cómo usar este curso Abre cada lección, lee la teoría y prueba los botones de las demos. Después abre las Herramientas de Desarrollador (tecla F12) y mira la consola: ahí también se imprime lo que hace cada ejemplo.
